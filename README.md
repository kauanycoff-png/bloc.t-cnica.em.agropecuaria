# Blog Técnica Agropecuária

Projeto de blog criado por **Kauany Coff**, com foco em temas da agropecuária.

## 🌱 Tema
Ovinocultura Orgânica

## 📚 Sobre o projeto
Este blog apresenta informações sobre a criação de ovinos em sistemas orgânicos, com foco em sustentabilidade, bem-estar animal e produção natural.

## 📌 Conteúdo
- Introdução à ovinocultura orgânica
- Manejo sustentável
- Benefícios ambientais e econômicos
- Cuidados com os animais
